# 大作业运行说明 Log

## 前端运行说明

  1. 在final目录下，`cd fe` 进入前端工程目录<br>
  2. `npm i` 安装依赖<br>
  3. `npm run build` 将资源打包到../server/static目录<br>

## 后端运行说明

  1. 在final目录下，`cd server` 进入后端工程目录<br>
  2. `npm i` 安装依赖<br>
  3. `tsc` 进行编译<br>
  4. `node dist/index.js` 开始运行<br>
  5. 在浏览器访问 `127.0.0.1:3001` 访问页面<br>
